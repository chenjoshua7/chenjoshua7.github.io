Text Classification

Special thanks to my wonderful project partners: Mathieu Breier, Carlos Perez, and Rafael Gallegos.


This was a text classification task of the this Hugging Face dataset from Valurank. There are 22,462 articles with 120 topics. We grouped the topics into seven groups and used a 50%, 25%, 25% train, validation, test split. As a part of this project, we played around with data augmentation and data generation using Generative AI. These were the final results on the testing set using our best models. Note that the baseline with Tf-Idf Vectorizer - Logistic Regression.

×

For reference, here is the results of the Validation Set. Note: The baseline classifier was trained on the entire set. However, Limited BERT, Augmented BERT, OpenAI Aug. BERT, and combined BERT were only trained on 32 data samples.

For Augmentation, we implemented non-deep learning approaches. We took the tail end of data as unique data points and implemented random deletion and random swap. ("EDA: Easy Data Augmentation Techniques for Boosting Performance on Text Classification Tasks" by Jason Wei and Kai Zou).

For Data Generation, we queried the Chat GPT API to generate 5 samples for each topic. Note that both methodologies produced large leaps in performance on limited data with the combination of the methods producing solid results on just the 32 samples.
