Article Classification with Fine-tuned DistilBERT

Using DistilBERT from Hugging Face to classify 22,462 news articles into 7 labeled categories. I used data augmentation and data generation techniques to boost performance by 20% on only 32 samples. Using the same techniques, I was able to reach 85.98% accuracy.
Skills: Natural Language Processing, Keras, Transformers, Transfer Learning, Data Augmentation, Model Distillation

Classifying Urban Land Covers with Linear Models

The distribution of a city's surface, known as land cover, significantly impacts flood risk and temperature extremes, making it crucial for public policy, especially amid the climate crisis. Classifying urban land cover types, such as trees and buildings, using aerial image analysis helps assess impervious surfaces, with input variables including brightness, color, and shape of objects.
Skills: Feature Selection, Regularization, Bayesian Model Averaging, Model Selection, R
